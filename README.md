# LifeFlow Fitness Deployment

Deployment repository for LifeFlow Fitness.

This repository contains:

- Docker Compose configuration
- Reverse proxy configuration
- Production deployment setup
- CI/CD deployment target

## Services

- Frontend (React + Nginx)
- Backend (Express API)
- MongoDB

## Deployment

Deployment is triggered automatically from the frontend and backend repositories through GitLab CI/CD pipelines.
